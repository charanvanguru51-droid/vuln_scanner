# 🔍 Network Vulnerability Scanner

A Python-based tool to scan networks, detect open ports, identify services, and highlight basic vulnerabilities.

## 🚀 Features
- Port scanning using Nmap
- Service detection
- Basic vulnerability identification
- HTML report generation
